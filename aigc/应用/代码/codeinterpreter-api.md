# codeinterpreter-api

LangChain实现的ChatGPT开源代码解释程序。CodeBoxes后端使用沙盒python代码执行。
CodeBox是LLM应用简单的云端基础设施。除了LLM使用OpenAI API Key其它均可本地执行。

* 数据集分析，股票图表，图像处理
* 互联网接入和自动Python包安装
* 输入 text + files -> 接收 text + files
* 会话记忆:根据之前的输入进行响应
* 除了OpenAI API (OpenOrca或其他很快到来)，所有内容都在本地运行。
* 使用CodeBox API在生产环境中轻松扩展(即将推出)
